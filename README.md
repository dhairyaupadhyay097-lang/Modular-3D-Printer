# Modular-3D-Printer
A custom-built DIY 3D printer with a 200×200 mm build area, designed for precision prototyping, robotics, and CubeSat development. The printer uses a rigid frame, PETG printed structural parts, and an all-metal hotend for reliable PLA, PETG, and ABS printing. Built to learn real engineering, motion systems, and advanced manufacturing from scratch.
