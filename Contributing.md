# Contributing to Atlas 3D Printer

Thanks for your interest in contributing to Atlas 3D Printer!

I would welcome contributions that improves the design, documentation, firmware, or overall build experience.

## Ways You Can Contribute

- Improve the CAD design
- Report bugs or design issues
- Suggest new features
- Improve documentation
- Correct spelling or grammar
- Optimize the Bill of Materials (BOM)

## Before You Start

Please check existing Issues before creating a new one.

## How to Contribute

1. Fork this repository.
2. Create a new branch.
3. Make your changes.
4. Test your changes if possible.
5. Commit with a clear message.
6. Open a Pull Request.

## CAD Guidelines

- Keep part names descriptive.
- Upload editable CAD files whenever possible.
- Export updated STEP files if the design changes.
- Do not delete existing parts unless necessary.

## Documentation Guidelines

- Use clear English.
- Include images wherever needeed.
- Keep instructions begginer friendly to follow.
